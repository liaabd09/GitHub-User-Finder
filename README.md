# GitHub User Finder

**Автор:** Абдуллина Лия

## Описание

GitHub User Finder — это десктопное приложение с графическим интерфейсом для поиска пользователей GitHub, просмотра их информации и сохранения избранных пользователей в локальном файле.

## Возможности

- Поиск пользователей GitHub по имени
- Отображение списка найденных пользователей с информацией (ID, количество репозиториев, подписчиков, ссылка на профиль)
- Добавление пользователей в избранное
- Просмотр списка избранных пользователей с датой добавления
- Удаление пользователей из избранного
- Автоматическое сохранение избранных пользователей в JSON-файл
- Валидация ввода (поле поиска не может быть пустым)
